# Driver Drowsiness Detection System

## Overview
Hardware-based Driver Droswiness Detetcion with ESP32-CAM it's a real-time Driver Monitoring System (DMS) 
developed to improve road safety by detecting signs of driver fatigue using computer vision techniques.
The system analyzes facial landmarks captured by an ESP32-CAM to monitor the driver's eye movements and calculates the Eye Aspect Ratio (EAR)
to identify prolonged eye closure 
a key indicator of drowsiness. When fatigue is detected, the system immediately triggers visual and audible alerts through an Arduino Uno, LEDs, buzzer, and LCD to warn the driver. Additionally,
a Flask-based web dashboard provides live video streaming and real-time system monitoring.
The project integrates software and hardware components into a low-cost prototype capable of
delivering early drowsiness detection and timely driver alerts.


## Features
- Real-time drowsiness detection
- Eye Aspect Ratio (EAR)
- ESP32-CAM live streaming
- Flask monitoring dashboard
- Arduino alerts (buzzer, LEDs, LCD)
- Multithreading
